myFlix

This is a REST API for an application called “myFlix” that interacts with a database that stores data about different movies. The REST API will be accessed via commonly used HTTP methods like GET, POST, PUT, DELETE. The application uses HTTP Authentication for the login and JWT Authentication for all endpoints.
